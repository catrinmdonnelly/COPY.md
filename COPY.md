## Banned phrases

<!-- THE highest-leverage section in this file. Research benchmarks:
     ~90% AI compliance with explicit vocabulary rules vs ~55% for
     vague principles. Long is good here. Update every time AI reaches
     for a phrase that isn't yours. -->

The single most important section. Research shows AI follows a banned-phrases list at ~90% compliance, versus ~55% for vague principles. Write the list. Update it weekly.

### Corporate jargon
| Banned | Use instead |
|--------|-------------|
| Leverage | Use |
| Synergy | Working together |
| Stakeholders | The people involved |
| Deliverables | What we'll send / give you |
| Bandwidth | Time / capacity |
| Alignment | Agreement |
| Ecosystem | The set of things around it |

### AI tells (vocabulary)
| Banned | Why |
|--------|-----|
| Delve | The single most-mocked AI word |
| Tapestry | Empty metaphor |
| Landscape | Empty metaphor |
| Foster | Corporate filler |
| Underscore | Just say "show" |
| Pivotal / Crucial | Almost never accurate |
| Multifaceted | Padding |
| Comprehensive | Almost always padding |
| Meticulous | Often padding |
| Vibrant | Empty descriptor |
| Testament | Empty connector |
| Robust | Empty descriptor |
| Seamless | Empty descriptor |
| Innovative | Empty descriptor |
| Solutions | Almost always padding. "Service" or "tool" usually fits |

### AI tells (structural)
- "It's worth noting that..." (cut, just say it)
- "In today's fast-paced world..." (cut, never recover)
- "Not just X, but Y" (overused, rephrase)
- Three-noun lists ("clear, concise, and comprehensive") (pick one)
- Headings in title case (use sentence case)
- "Navigate the [adjective] landscape" (banned outright)
- Em dashes used as the default connector (use commas or full stops)

### Hype and superlatives
- Game-changing, revolutionary, transformative
- Unlock, supercharge, disrupt
- World-class, best-in-class, cutting-edge

### Fake urgency / pressure
- Limited spots available
- Don't miss out, last chance
- Act now, today only

### Hedged / weasel language
- "Industry experts agree"
- "Studies show" (cite the study or cut it)
- "Many believe"
- "May help to" (will it or won't it)

### Salesy openers
- "I hope this email finds you well"
- "I'd love to pick your brain"
- "Let's touch base / circle back / deep dive"
- "Quick favour"

### Empty flattery
- "I love what you're doing" without a specific reason
- "Huge fan of your work" without naming the work

### Disclaimers dressed as authenticity
- "No sales pitch"
- "Genuinely just curious"
- "Promise I'm not selling"

If you have to tell someone you're not selling, the copy already sounds like selling.

### Generic CTAs
- "Learn more"
- "Get started today"
- "Click here"

Replace with what the action actually does. "Read the guide." "Book a 30-minute call." "Download the spreadsheet."

### AI self-narration
- "Certainly!"
- "Great question!"
- "Here's a breakdown..."
- "I'd be happy to help."

(These are for chat replies, not user-facing copy. But the AI tries to write copy this way too.)

## Self-check before sending

<!-- The checklist that closes the loop. Run every draft through this
     before publishing. The act of running the check is what catches
     drift. -->

Run every draft through this list before publishing.

- [ ] No em dashes anywhere
- [ ] No banned phrases (scan against the list above)
- [ ] No staccato fragments. Sentences are full and natural.
- [ ] Sentence case for headings, buttons, labels
- [ ] If I read this aloud, would a real person say it?
- [ ] If a friend who knows me read this, would they guess it was AI?
- [ ] Is the CTA specific (not "learn more")?
- [ ] If this is salesy, have I deleted the disclaimers about not being salesy?
- [ ] Region is right (British vs American English)
- [ ] If it's outreach, does it open with who, why, and what's in it for them?
